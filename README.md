# worldcup-soccer

[![Built with Periplum](https://img.shields.io/badge/built_with-Periplum-4da3ff)](https://periplum.js.org)

The men's football World Cup finals plotted at each host city and coloured by host
continent. Popups list the host, final venue, and winner. Press play to watch it hop
around the globe, or drag the date slider.

## Data & updates

`data.json` is hand-curated — one entry per tournament. It changes only every four years,
so there is **no script and no scheduled job**: after each final, add the host city and
winner to `data.json` and commit.
